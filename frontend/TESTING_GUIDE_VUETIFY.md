## 💡 提示

### 如何查看主题是否正确切换？

打开浏览器控制台（F12），执行：
```javascript
// 查看当前主题名称
console.log(document.documentElement.getAttribute('data-theme'))

// 或者查看 Vuetify 实例
console.log(document.querySelector('.v-application').classList)
```

### 如何快速切换前后台测试？

在控制台执行：
```javascript
// 跳转到后台
window.location.href = '/admin/photos'

// 跳转到前台
window.location.href = '/'
```

### 如何测试自定义光标？

在前台页面移动鼠标：
- 应该看到金色圆圈光标
- 应该看到淡金色光晕
- 悬停在按钮/链接时光标放大
