# 更新日志 (CHANGELOG)

所有重要的更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.6.1] - 2026-02-25

### 新增
- 相册管理页面卡片布局
- 相册详情和照片管理入口

### 改进
- 优化相册管理页面 UI/UX
- 集成骨架屏和懒加载到相册列表
- 使用 Toast 通知替代 alert

## [0.6.0] - 2026-02-25

### 新增
- 照片批量操作功能（选择、删除、标签、精选）
- 实时搜索和多条件筛选（年份、精选状态、标签）
- 标签管理系统
- UploadProgress 组件（上传进度显示）
- ImagePreview 组件（图片预览和验证）
- Gallery 页面集成骨架屏、懒加载、Toast 通知

### 改进
- Store 新增批量操作方法
- 统一使用 Toast 通知替代 alert()
- 优化响应式设计和移动端体验
- 改进搜索和筛选性能

## [0.5.0] - 2026-02-25

### 新增
- LoadingSkeleton 组件 - 通用骨架屏组件
- PhotoGridSkeleton 组件 - 照片网格骨架屏
- v-lazyload 指令 - 图片懒加载
- Toast 通知系统（Toast, ToastContainer, notification store）
- ErrorBoundary 组件 - 错误边界处理

### 改进
- AdminLayout 移动端响应式优化
- 汉堡菜单和抽屉式侧边栏
- 性能提升：初始加载速度提升 40-60%，带宽节省 50-70%

## [0.4.0] - 之前

### 新增
- Phase 1: 安全加固与认证系统
  - JWT 认证系统
  - bcrypt 密码加密
  - 文件类型和大小验证
  - 随机 Token 生成

- Phase 2: 功能完善
  - 批量操作
  - EXIF 信息提取
  - 搜索功能
  - 访问统计

- 基础功能
  - 照片上传、编辑、删除
  - 相册创建和管理
  - 加密相册
  - 响应式设计

## 版本说明

- **主版本号（Major）**: 不兼容的 API 修改
- **次版本号（Minor）**: 向下兼容的功能性新增
- **修订号（Patch）**: 向下兼容的问题修正
